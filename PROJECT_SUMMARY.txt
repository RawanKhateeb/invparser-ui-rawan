API CONFIGURATION
================================================================================

Base URL: http://localhost:8080

Endpoints:
1. POST /extract
   - Upload invoice file
   - Body: multipart/form-data (file)
   - Response: { invoiceId, vendor, amount, date, status }

2. GET /invoice/{id}
   - Fetch invoice details by ID
   - Response: { invoiceId, vendor, amount, date, items[], status }

3. GET /invoices/vendor/{vendor_name}
   - Search invoices by vendor
   - Response: { invoices: [], count: number }

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Cannot connect to backend"
Solution: 
  1. Verify backend running: http://localhost:8080
  2. Check network connectivity
  3. Verify API_BASE_URL in lib/api.ts

Problem: "Login not working"
Solution:
  1. Verify credentials: admin/admin
  2. Check localStorage enabled in browser
  3. Try incognito/private mode
  4. Clear browser cache

Problem: "Upload fails"
Solution:
  1. Check file format (PDF, JPEG, PNG, GIF, WebP)
  2. Check file size < 50MB
  3. Verify /extract endpoint is working
  4. Check browser console for errors

Problem: "Page won't load"
Solution:
  1. Check dev server running (npm run dev)
  2. Verify no build errors
  3. Clear browser cache
  4. Check browser console
